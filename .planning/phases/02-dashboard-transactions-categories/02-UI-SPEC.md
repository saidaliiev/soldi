## Typography

All presets from `TYPE` in `apps/mobile/src/design/typography.ts`. Use preset name, never inline values.

### Type Scale Architecture

Phase 2 uses **4 active text sizes** in the phase type scale: **40pt, 28pt, 16pt, 14pt**.

`TYPE.displayXL` (64pt Oswald) is a **display singleton** — it appears on exactly one element on one screen (MonthlyTotalHero on DashboardScreen). It is deliberately excluded from the phase type scale count. Its presence does not add a fifth scale step; it is a one-off brand statement, architecturally equivalent to a logotype.

The **3-typeface system** (Oswald display / EB Garamond editorial / Manrope UI) is mandated by `CLAUDE.md`. Each typeface uses at most 2 weights: Oswald 500; EB Garamond 400 regular + 600 semibold italic; Manrope 500 medium + 600 semibold. The apparent weight variety is a structural consequence of a deliberate 3-typeface architecture, not weight proliferation — each individual typeface stays within its 2-weight budget.

`TYPE.editorialLead` (20pt EB Garamond) exists in `typography.ts` but is **excluded from Phase 2** — it is not used in any component. Empty-state phrases that previously referenced it are reassigned to `TYPE.displayM` (28pt Oswald italic) for display-weight impact or `TYPE.editorialBody` (16pt EB Garamond italic) for softer editorial tone. See per-component assignments below.

`TYPE.uiMeta` (12pt Manrope) exists in `typography.ts` but is **excluded from Phase 2** — all roles previously assigned to it are reassigned to `TYPE.uiLabel` (14pt Manrope) for improved legibility at small UI sizes. The only exception is tab bar labels which historically used 12pt — they are also raised to `TYPE.uiLabel` (14pt) for consistency.

## Animation Contract

All animations use `react-native-reanimated` v4. Worklets used only for swipe gesture (D-26).

| Interaction | Curve | Duration | Notes |
|-------------|-------|----------|-------|
| Donut slice interpolation on month change | `withTiming(value, { easing: Easing.out(Easing.cubic) })` | 300ms | D-05; arc path values interpolate via shared values |
| Donut center label morph (tap slice → show category) | `withTiming` opacity 0→1 + translateY 4→0 | 200ms | D-04; outgoing label fades out simultaneously |
| Donut center label morph (untap → restore total) | `withTiming` opacity 0→1 + translateY 4→0 | 200ms | Same curve |
| Month swipe transition | `withTiming(targetOffset, { easing: Easing.out(Easing.quad) })` | 250ms | D-02; horizontal translateX |
| Transaction row swipe-left reveal | worklet `useAnimatedGestureHandler` — tracks pan, clamps to −120pt | gesture-driven | D-11; no withTiming during active gesture |
| Transaction row swipe snap to open | `withSpring(−120, { damping: 20, stiffness: 200 })` | spring | Triggered when pan > 60pt threshold |
| Transaction row swipe snap to close | `withSpring(0, { damping: 20, stiffness: 200 })` | spring | On release below threshold or after category selection |
| Bottom sheet mount (recategorize / editor) | `@gorhom/bottom-sheet` internal spring | default gorhom | D-11, D-17; do not override gorhom animation |
| Filter modal appear | `withTiming` opacity 0→1, screen slides up | 220ms `Easing.out(Easing.cubic)` | Full-screen modal (D-13) |
| Filter modal dismiss | `withTiming` opacity 1→0, screen slides down | 180ms | Faster exit |
| Drag-drop lift in category editor | `withSpring` scale 1→1.04 + shadow deepens | spring, damping 15 | D-19; haptic medium fires on long-press |
| Empty-state entrance | `withDelay(100, withTiming)` opacity 0→1 + translateY 8→0 | 300ms | Claude's Discretion default |
| Category selection haptic | `Haptics.impactAsync('light')` | — | D-12 |
| Drag-drop long-press haptic | `Haptics.impactAsync('medium')` | — | D-19 |

## Component Specifications

### DashboardScreen

**Route:** `app/(tabs)/index.tsx` (replaces Phase 1 skeleton)
**Layout:** Single `ScrollView` (not FlatList — content is not virtualized list), horizontal padding `SPACING.md` (16pt), top padding `SPACING.xl` (32pt) + safe area inset.
**Scroll:** Vertical, no bouncing clamp on the month swiper (month swiper handles its own horizontal gesture).
**Data load:** `useFocusEffect` → `executeSync` queries for selected month (D-27). Synchronous from op-sqlite; no loading spinner on subsequent focuses if data is cached in Zustand.
**Loading state (initial mount):** Skeleton shimmer on hero number and donut area — `COLORS.surface` rectangles animating opacity 0.4↔0.8 with `withRepeat(withTiming(..., 400ms))`. Duration max 300ms before data arrives (D-27).
**Error state:** Inline error below hero: `TYPE.uiLabel` in `COLORS.error` — `"Couldn't load data. Tap to retry."` (`dashboard.error_load`)
**Vertical section order (D-01):**
1. MonthSwiper (month navigation header)
2. MonthlyTotalHero (Oswald displayXL total)
3. DonutChart + CategoryBreakdownRows (same visual block)
4. DigestCard ("yesterday in money")
5. TopCategoryRows (top 5, below digest)
**Section gaps:** `SPACING.lg` (24pt) between each block.
**Accessibility:** `accessibilityRole="none"` on scroll container; each section has `accessibilityLabel` as described per component below.

### MonthSwiper

**Purpose:** Horizontal swipe to navigate between months (D-02).
**Range:** Month of earliest transaction in DB → current month + 1 (inclusive).
**Gesture:** `react-native-gesture-handler` `PanGestureHandler`, horizontal only. Worklet tracks offset; snaps to nearest month on release.
**Visual:** Month label centered — `TYPE.displayM` (`COLORS.textPrimary`). Left/right chevron arrows (custom SVG, 20×20pt, `COLORS.textSecondary`). Chevron hidden at bounds (past lock / future lock).
**Future month placeholder:** Label renders normally; dashboard body shows `EmptyState` variant "future" — no hero total, no donut. Copy: `"No data yet."` + `"Add a transaction to see this month."` + CTA `"Add transaction"`.
**Accessibility:** `accessibilityLabel="[Month] [Year], swipe to navigate"`, `accessibilityRole="adjustable"`, `accessibilityActions=[{name:'increment'},{name:'decrement'}]` mapped to next/prev month.

### DonutChart

**Purpose:** Skia arc chart showing category breakdown (D-04, D-05, D-06).
**Implementation:** Pure Skia `Path` arcs (no Victory Native). Roll arc math from category percentages. Canvas size: 200×200pt (Claude's Discretion — fits above-the-fold on SE 2020 width 375pt minus 2×16pt padding = 343pt; chart is 200pt centered or left-aligned with category legend to the right).
**Stroke:** 10pt width, round cap (`StrokeCap.Round`), gaps between slices 2pt (achieved by trimming arc start/end by 1pt each side in angle space).
**Slice colors:** Use category's assigned color from the D-22 swatch set. "Other" slice color: `COLORS.textMuted`.
**Slice order:** Descending by absolute amount (D-06). Top 5 slices explicit; remainder → "Other".
**Center label — default (no tap):** Monthly total in `TYPE.displayL` (Oswald 40pt) color `COLORS.textPrimary`, centered. `TYPE.uiLabel` `"Total"` label above in `COLORS.textMuted`.
**Center label — on slice tap (D-04):** Category name `TYPE.displayM` (28pt) top, amount `TYPE.displayL` (40pt) middle, percentage `TYPE.uiLabel` (14pt) `COLORS.textSecondary` bottom. Morphs with 200ms opacity/translateY transition. Tap anywhere outside slice to reset.
**Tap target:** Each slice arc region ≥ 44pt arc length. Thin slices (<5% of total) merged visually into adjacent or "Other" for tappability.
**Animation on month change:** All arc paths animate from previous month values to new month values via `withTiming(300ms, Easing.out(Easing.cubic))` shared values (D-05).
**Performance:** First frame target < 100ms from dashboard mount on populated DB (QUAL-06, D-27). Skia handles GPU render; data query is synchronous `executeSync`.
**Accessibility:** `accessibilityLabel="Spending donut chart. [Top category] [amount], [percentage of total]. Double-tap a slice to see details."`, `accessibilityRole="image"`. Each tapped slice announces: `"[Category]: [amount], [percentage]"` via `accessibilityLiveRegion="polite"`.

### DigestCard

**Purpose:** "Yesterday in money" editorial card (D-07, D-08).
**Container:** Full-width card, `RADIUS.lg` (16pt), `COLORS.surface` background, `SHADOWS.card` shadow. Internal padding `SPACING.md` (16pt) all sides.
**Layout (top to bottom):**
1. Prefix label: `"yesterday in money"` — `TYPE.uiLabel` (`COLORS.textMuted`), `mb: SPACING.xs` (4pt)
2. Yesterday total: `TYPE.displayL` (40pt Oswald) `COLORS.accent` (expense color). `fontVariant: ['tabular-nums']`.
3. Sparkline: 32pt height, full card width minus 2×16pt padding. Skia `Path` single line, no fill, `COLORS.accent`, anti-aliased. Data = last 7 daily totals (D-07).
4. MoM compare phrase: `TYPE.editorialBody` italic (`COLORS.textSecondary`). Example: `"−€42 vs last month's daily average"`. `mt: SPACING.sm` (8pt).
**Sparkline data:** 7 data points (day −6 through yesterday). If a day has no spend, value = 0. X axis evenly distributed across width. Y axis auto-scales to data range (min 0, max = highest day). No axis lines, no labels on sparkline.
**Editorial tone for MoM phrase:** Declarative, slightly literary. Templates:
- Spent more: `"+ €{delta} above last month's daily average"` (`dashboard.digest_above_avg`)
- Spent less: `"− €{delta} below last month's daily average"` (`dashboard.digest_below_avg`)
- Same: `"Right on track for the month."` (`dashboard.digest_on_track`)
- No prior month data: `"First month tracked."` (`dashboard.digest_first_month`)
**Accessibility:** `accessibilityRole="none"` on card, inner elements labeled: yesterday total `accessibilityLabel="Yesterday total: [amount]"`, MoM phrase read as static text, sparkline `accessibilityLabel="Spending sparkline, last 7 days"`.

## Performance Contract

| Metric | Target | Approach |
|--------|--------|----------|
| Dashboard first frame | < 100ms from mount | `useFocusEffect` + `executeSync` (synchronous DB read, no await); Skia GPU render |

| Search debounce | 150ms input delay (D-15) | `useRef` debounce timer, not lodash |

| Swipe gesture | 60fps during active swipe | Worklet — runs on UI thread (`useAnimatedGestureHandler`) |
